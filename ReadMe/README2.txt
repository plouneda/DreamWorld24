DreamForge Unit Testing README

To run the unit tests for the DreamForge project, follow these steps:

1. Ensure that you have Python installed on your system.

2. Navigate to the project directory containing the unit test file (test_dreamforge.py).

3. Run the following command to execute the unit tests:

   python -m unittest test_dreamforge.py

4. View the test results in the console output. Each test case will be indicated as either OK (passed) or FAIL (failed).

5. Analyze any failed tests and make necessary adjustments to the project code.
