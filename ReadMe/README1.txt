]DreamForge Project README

To run the DreamForge project, follow these steps:

1. Clone the DreamForge repository from GitHub:

   git clone https://github.com/plouneda/DreamWorld24.git

2. Navigate to the project directory:

   cd DreamWorld24

3. Install the required dependencies using pip:

   pip install -r requirements.txt

4. Run the main application file:

   python dreamforge.py

5. Access the DreamForge platform through your web browser at http://localhost:8000

6. Enjoy crafting, exploring, and experiencing the realm of dreams with DreamForge!
